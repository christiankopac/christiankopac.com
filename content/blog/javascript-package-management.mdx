## Classic Package Managers

- https://yarnpkg.com/
- https://www.npmjs.com/

## Alternatives to the classics

- https://www.skypack.dev/
  Load optimized npm packages with no install and no build tools.

- https://pnpm.io/
  Fast, disk space efficient package manager

- https://jspm.org/
  jspm provides a module CDN and package management for import maps, allowing any package from npm to be loaded directly in the browser fully optimized without further tooling.
